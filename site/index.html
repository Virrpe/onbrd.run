How it works

Run it free in your browser. Upgrade later if you need more checks or automation.

1) Install & run

Install the Chrome extension and open your onboarding flow. Click "Audit" — it runs locally in the browser.

2) Review checks

See a 0–100 score with prioritized checks across signup friction, activation path, copy/clarity, momentum, and technical basics.

3) Share or iterate

Export an HTML report and share it as-is. Re-run after fixes to track improvements.

Prefer automation? The API mirrors the checks for CI or scheduled audits. Start with the extension, move to API when you need scale.
